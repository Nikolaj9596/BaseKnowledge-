# Figma Hot Key

## Common Hot Key

**Ctrl +** - Маштабировать элемент
**Shift + Left Mouse** - Выделить несколько элементов
**Ctrl + Left Mouse** - Выделить в группах элемент
**Alt** - Зажать показывает растояние до элементов

## Groups

**Ctrl + G** - Сгруппировать
**Ctrl + Shift + G** - Разломать

## Frame

**A** or **F** - Создать пучтой фрейм
**Ctrl + Alt + G** - Обернуть фреймом
**Shift + A** - Фрейм с фвтолейфутом
**Ctrl + Shift + G** - Разломать

**Alt + H** - Выровнять по горизонтали
**Alt + V** - Выровнять по вертикали
**Alt + W/A/S/D** - Быстро поставить к границам фрейма

## Section

**Shift + S** - Обернуть в секцию
**Ctrl + Shift + G** - Разломать
